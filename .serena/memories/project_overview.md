# Law-fi Project Overview

**Project Name**: Law-fi
**Target Audience**: 하이엔드 법조인 (변호사) 및 로스쿨 학생
**Core Concept**: 익명 기반의 프리미엄 전문직 라운지
**Design Theme**: Aerocano (에스프레소의 묵직함과 크레마의 부드러움 대비, Clarity & Stealth)
**Tone**: Minimalist, Professional, Relaxed

## User Roles
* ADMIN: 시스템 관리 및 사용자 인증 승인 권한
* LAWYER: 인증된 변호사
* STUDENT: 인증된 로스쿨생
* GUEST: 가입 후 인증 절차를 완료하지 않은 상태
