# Law-fi Tech Stack

* **Monorepo Manager**: Turborepo
* **Package Manager**: pnpm (v9)
* **Frontend**: Next.js 15 (App Router) with React Compiler
* **Admin/Backoffice**: Refine
* **Styling**: Tailwind CSS (Semantic Tokens only), shadcn/ui, Framer Motion
* **State Management**: TanStack Query v5 (Server), Zustand (Client)
* **Backend Data & Auth**: Supabase (PostgreSQL with RLS, Auth via Email/OTP, Storage for Private Buckets)
* **ORM**: Prisma
* **Validation**: Zod
